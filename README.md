1. 项目名称  
逻辑说—— 自然语言问卷逻辑生成器

2. 背景与痛点  
在做学生工作、社团招新、课程调研时，经常需要设计带逻辑跳转的问卷（例如：根据年级、经验跳转到不同题目组）。现有工具（问卷星、腾讯问卷等）虽然支持逻辑设置，但存在以下问题：
设置路径深，需要点击多处菜单
学习成本高，普通用户不知道“逻辑跳转”这个功能在哪里
修改需求时，需要逐个题目重新配置，耗时易错
本人作为班干部，曾多次被此类需求折磨，深感做问卷的时间本应花在“想问题”上，而不是“找选项”。

3. 目标用户  
初级用户：学生会/社团干事、班干部，需要快速制作招新、活动调研/反馈问卷  
扩展用户：企业HR、教师/教育机构、小型调研人员、内容创作者，需要处理复杂问卷

4. 核心价值  
把“做问卷”变成“说需求”。
用户用自然语言描述问卷逻辑（如“大一新生跳转基础题，有经验的直接问实操”），系统自动生成包含题目、选项、跳转逻辑的问卷结构，并导出到目标平台，用户只需微调即可发布。

5. 功能列表  
一周版本
需求输入：用户通过文本框输入自然语言需求 
问卷结构生成：调用 AI，输出包含题目、选项、逻辑跳转的 JSON 结构 
结构预览：以人类可读的文本形式展示问卷结构（题目、选项、跳转条件） 
多轮修改：用户可继续输入自然语言修改指令（如“把第二题改成单选题”），AI 更新问卷结构 
导出文件：将最终结构转换为目标平台（如腾讯问卷）的导入格式（JSON），提供下载 
使用指引：提供图文教程，指导用户如何将文件导入腾讯问卷并发布

二期（待定）
一键发布：通过 API 直接在目标平台创建问卷并返回填写链接 
可视化预览：以图形化界面展示问卷结构，支持点击修改 
模板库：保存常用问卷结构（招新、反馈、报名）供用户复用 

6. 用户操作流程  
用户打开网页，看到一个大文本框，提示“描述你的问卷需求，例如：我是学生会负责人，要招募志愿者，大一新生跳转基础信息，有经验的同学直接问实操经验”

用户输入需求，点击“生成问卷”

系统显示生成的问卷结构（文本预览）

用户查看，如果不满意，继续在输入框输入修改指令（如“把第二题改成单选题，选项为‘是/否’”），点击“修改”

用户点击“导出”，下载 JSON 文件

用户根据页面上的图文教程，将文件导入腾讯问卷，发布
